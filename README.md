# Clojure Sandbox

> Clojure POCs, tests

- https://clojure.org/

> Clojure is a modern, functional Programming Language on the Java Virtual Machine (JVM).

## Setup

* Prerequisite: Java (version >= 8)

* Install Clojure on MacOS
```
brew install clojure/tools/clojure
```

## Usage

- Start REPL
```shell
clj
```

- Run unit tests
```shell
clj -X:test
```
